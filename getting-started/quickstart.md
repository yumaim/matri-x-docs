# クイックスタート

**3分でMatri-Xを使い始める**

---

## ステップ1: Matri-Xにアクセス

[https://matri-x-algo.wiki](https://matri-x-algo.wiki) をブラウザで開きます。

<figure><img src="images/lp-toppage.png" alt="Matri-X トップページ"><figcaption>Matri-X トップページ — 右上の「無料で始める」からアカウント登録に進めます</figcaption></figure>

---

## ステップ2: アカウント登録

トップページ右上の **「無料で始める」** ボタンをクリックすると、アカウント登録ページに遷移します。

<figure><img src="images/lp-register-click.png" alt="無料で始めるボタンをクリック"><figcaption>「無料で始める」ボタンをクリック</figcaption></figure>

登録ページで以下を入力します:
1. **お名前**（表示名）
2. **メールアドレス**
3. **パスワード**（8文字以上）
4. 「登録する」をクリック

または **「Googleで登録」** ボタンでワンクリック登録も可能です。

<figure><img src="images/register-page.png" alt="アカウント登録ページ"><figcaption>アカウント登録ページ — Google認証またはメール登録が選べます</figcaption></figure>

💡 **完全無料** — クレジットカード登録は不要です

詳しくは → [アカウント登録ガイド](signup.md)

---

## ステップ3: ログイン

登録済みの方は、ログインページからメールアドレスとパスワードでサインインします。

<figure><img src="images/login-page.png" alt="ログインページ"><figcaption>ログインページ — Googleアカウントまたはメールアドレスでログイン</figcaption></figure>

---

## ステップ4: ダッシュボードを確認

ログイン後、ダッシュボードが表示されます。左側の **サイドバー** から各機能にアクセスできます。

<figure><img src="images/dashboard-home.png" alt="ダッシュボードホーム"><figcaption>ダッシュボードホーム — サイドバーから各機能へアクセス</figcaption></figure>

### サイドバーの主要メニュー

| メニュー | 説明 |
|---------|------|
| 📊 ダッシュボード | ホーム画面 |
| 🔍 パイプライン探索 | アルゴリズム全体フローを可視化 |
| 📈 エンゲージメント分析 | 重み付けスコアの確認 |
| 🎮 シミュレーター | 投稿パフォーマンス予測 |
| 💬 フォーラム | コミュニティ |
| 🎫 チケット | 機能リクエスト・バグ報告 |
| 🔔 通知 | アクティビティ通知 |
| 👤 プロフィール | アカウント設定 |
| ⚙️ 設定 | 通知・プランの管理 |

---

## ステップ5: フォーラムで交流

登録後、まずは**フォーラム**でコミュニティに参加しましょう。

- 質問・相談
- ベストプラクティスの共有
- 最新トレンドの議論
- 成功事例の紹介

詳しくは → [フォーラムの使い方](../guides/forum.md)

---

## ステップ6: 通知と設定

### 通知ページ
コメントへの返信やチケットの更新がリアルタイムで届きます。

### 設定ページ
通知の ON/OFF、メールアドレスの変更、プラン確認ができます。
